Optimize your VPN with Cloudflare's global network. Follow these enterprise-grade configurations:

🌐 DNS Configuration

A record: yourdomain.com → YOUR_VPS_IP CNAME: *.yourdomain.com → yourdomain.com
  • ✅ Proxy status: Orange cloud for HTTP, Gray for HTTPS
  • ✅ TTL: Auto (recommended for dynamic IPs)
  • ✅ Enable DNSSEC for enhanced security

⚙️ Performance Settings

  • 🚫 Rocket Loader: OFF (breaks WebSocket)
  • ✅ WebSocket: ON (essential for tunneling)
  • 🚫 Always Use HTTPS: OFF (protocol flexibility)
  • 🚫 Under Attack Mode: OFF (blocks legitimate traffic)

  • ✅ HTTP/2: ON for better performance

  • 🔧 Security Level: Medium (adjust as needed)

  • 🚫 Challenge Passage: 30 minutes
  • ✅ Browser Integrity Check: ON
